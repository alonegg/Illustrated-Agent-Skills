插图位置：第 6 章 - 开头故事 - 配图 4
本段内容摘要：作者把自己做的技能分享给三个朋友，结果每个人都遇到了不同的问题——多语种不支持、缺 API Key、文件格式不兼容。在自己这里跑得好好的，到别人那里就全是问题。
读者要获得的“瞬间理解”：自己能用 ≠ 别人能用，隐含假设是技能分享的最大陷阱

配图作用：概念具象化

画面左右分栏，中间有黑色细分隔线。

【左半 - 在我这里】
黑色低透明度（浅灰调）作为底色区域。

画面中央：一个涂鸦小人坐在电脑前，表情开心，竖起大拇指。电脑屏幕上画一个大大的勾（✓）。小人头顶有一个小对话泡，里面是一颗小星星，表示满意。

电脑周围环境整洁：桌面上只有电脑和一杯咖啡，一切井然有序。

上方标签：“在我这里”

【右半 - 到别人那里】
\#D87757（赤陶色）作为底色区域。

画面中有三个涂鸦小人（和左边是同一个小人造型），分别遇到不同问题：

* 第一个小人对着电脑，屏幕上显示乱码般的符号，头顶冒出一个问号（？）—— 代表多语种不支持
* 第二个小人对着电脑，屏幕上显示一把钥匙加一个叉号（✗），小人双手摊开表示无奈 —— 代表缺 API Key
* 第三个小人对着电脑，屏幕上显示一个文件图标加一个叉号（✗），小人挠头 —— 代表文件格式不兼容

三个小人周围散落着叉号（✗）和问号（？），整体凌乱、有挫败感。

上方标签：“到别人那里”
