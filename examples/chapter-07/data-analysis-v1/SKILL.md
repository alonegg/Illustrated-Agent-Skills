# 数据分析

用户上传数据文件后，执行以下步骤：

## 步骤
1. 用 pandas 读取文件（支持 .xlsx、.xls、.csv）
2. 输出数据概览：行数、列数、各列数据类型
3. 数值列统计：均值、中位数、最大值、最小值、标准差
4. 检查缺失值和异常值
5. 用中文总结 3~5 条关键发现

## 注意
- 所有输出使用中文
- 数字保留两位小数
- 如果数据有明显的分组列，按组做对比分析
